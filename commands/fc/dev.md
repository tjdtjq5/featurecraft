# /fc:dev — 플러그인 개발/배포

featurecraft 플러그인을 수정하고 배포합니다.

## 사용 예제
```
/fc:dev status              → 현재 버전, 스킬 목록, 변경 사항 확인
/fc:dev edit ft-build       → ft-build 스킬 수정
/fc:dev add 새스킬명        → 새 스킬 추가
/fc:dev release             → 버전 올리기 + 커밋 + push
```

스킬 경로의 SKILL.md를 참조하여 실행하세요.
